# Alpaca Office Request - Camunda 8 enablement
 This repo contains process application resources, exercise descriptions and sample documents for the enablement session. 
 
 Attendees also need to clone/download this repository to get started with JobWorkers: [blank-demo ](https://github.com/McAlm/blank-demo)

 ## Preparation
 1. Create a Camunda 8 cluster version 8.9
 2. Create a WebModeler Project "Alpaca Office Request"
 3. Drag&Drop ./resources/Alpaca IDP.json to the root of the new project
 4. Create a new ProcessApplication "Alpaca Office Request"
 5. Delete the newly created process
 6. Drag&Drop all resources from ./resources/Office Alpaca Request" into the process application folder
 7. Clone/Download the repository https://github.com/McAlm/blank-demo
